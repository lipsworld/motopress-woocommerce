Extend MotoPress Content Editor plugin with WooCommerce shortcodes.

== Description ==

This plugin adds WooCommerce shortcodes to MotoPress Content Editor. By using it you are able to edit the layout of your online store visually, add products to your page and feature them the way you like. The plugin allows you to set how many products you want to display, how many columns you want to use and how to sort them. You also have an opportunity to show related items, recent, top rated or best selling products.

Use MotoPress along with the WooComerce and create a beautifully structured online shop.
